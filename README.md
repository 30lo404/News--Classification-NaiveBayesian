# News-Classification-NaiveBayesian

基于朴素贝叶斯的新闻文本分类项目实习

1.项目目标：
通过对朴素贝叶斯算法（Naive Bayesian Algorithm）的学习与代码实现，完成对来源于20个不同分组的20,000份新闻的识别与分类，进而了解机器学习背后的数学理论基础以及目前Machine Learning & Computer Version 的主要研究方向与实际应用。

2.项目计划：
(1)学习了解分布式版本控制工具GIT，并用GIT来管理协调机器学习项目进度。
(2)熟知机器学习的背景与实际的应用场景，对目前一些主流算法（CNN）的原理进行一定的掌握。
(3)了解掌握并自己实现朴素贝叶斯算法。
(4)将20,000份新闻一半用作训练集，一半用作测试集，通过算法完成新闻的识别与分类，统计识别的成功率。
(5)总结分析：是否可以通过对新闻文本的一些预处理（eg. 去除掉无关的标点符号与乱码）来提高成功率。
(6)探索Computer Version 图像视频智能识别领域的一些场景与算法，对AI算法行业有一些了解。
